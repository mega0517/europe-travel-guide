2. **알프스 2도시(잘츠부르크·인터라켄) 더위 대비** — 에어컨 미보장이 기본. 7월 폭염 가능성을 감안해, 더위에 민감하면 (a)선풍기 제공 여부 확인, (b)에어컨 보유 소수 매물 우선 검색, (c)잘츠부르크는 확정 호텔 활용을 고려.
3. **차량 동선 = 주차 + 진입 제한** — 밀라노 ZTL, 피란 구시가 진입 제한이 핵심 변수. "전용/인근 주차 명시 + 제한구역 밖" 매물을 우선하라.
4. **개별 가격·평점 비교 필요** — 본 검토는 조건 정합성 중심이다. 도시별 위시리스트(가이드의 "🤝 함께 편집하기" 링크)에서 4인이 함께 보며 가격·평점으로 최종 1곳을 좁히길 권한다.
5. **검토→재추천 루프** — 향후 일정/날짜가 또 바뀌면 예약가능 재검증이 필요하다. "인터라켄 재큐레이션" 사례처럼, 날짜 변경 시 같은 기준으로 재검토를 요청하면 부적합 숙소를 자동 교체한다.

## 예약 전 확인 체크리스트 (도시 공통)

- [ ] **침실 구성**: 침실 2개가 각각 더블/퀸인지(2쌍 부부 독립). "2 beds"가 아니라 "2 bedrooms"인지 재확인.
- [ ] **주차**: 무료 자체 주차인지, 인근 유료 주차인지(요금·거리). 진입 제한구역(밀라노 ZTL·피란 구시가) 여부.
- [ ] **에어컨**: 밀라노·피란은 필수 확인. 잘츠부르크·인터라켄은 폭염 대비로 선풍기/통풍 확인.
- [ ] **냉장고/주방**: 장기(인터라켄 3박)는 세탁기까지 확인.
- [ ] **숙박일 예약 가능**: 본 검토 시점엔 전부 가능. 예약 직전 해당 날짜로 "예약하기+총액"이 뜨는지 최종 확인(최소숙박 규정).
- [ ] **체크인 방식**: 늦은 도착(이동일) 대비 셀프 체크인/락박스 여부.
- [ ] **가격·평점**: 4인 1박 환산 합리성, 평점 ≥4.7·리뷰 수 충분한지.
